### STM32F103RBT6 外设驱动与模块化实验合集

本项目基于 **STM32F103RBT6** 微控制器，涵盖从基础入门到常用外设驱动的完整实验代码。每个工程独立配置，使用标准库、HAL库（CubeMX）以及RT-Thread等不同开发方式，适合嵌入式初学者系统学习或作为快速开发参考。

#### 主要模块与功能

| 实验编号 | 名称            | 核心内容                                             |
| :------- | :-------------- | :--------------------------------------------------- |
| 01~02    | test / led      | 环境测试、GPIO 输出控制 LED                          |
| 03~04    | buzzer / music  | 蜂鸣器驱动、简易音乐播放                             |
| 05       | button          | 按键输入检测（轮询/中断）                            |
| 06~07    | fwlib / cubemx  | 标准外设库与 CubeMX 配置对比                         |
| 08       | rtt             | 基于 RT-Thread 的工程示例                            |
| 09~10    | bitband / delay | 位带操作、SysTick 精准延时                           |
| 11       | ldt             | 自定义功能模块（可扩展）                             |
| 12~13    | dht / ds18b20   | 单总线温湿度传感器（DHT11）与 DS18B20 温度传感器驱动 |
| 14       | eint            | 外部中断控制                                         |
| 15~16    | iwdg / wwdg     | 独立看门狗与窗口看门狗实验                           |
| 17       | usart1          | 串口通信（发送/接收/中断）                           |
| 18       | gpio_iic        | 软件模拟 IIC，驱动常见 I2C 器件                      |
| 19       | oled            | OLED 显示屏驱动（IIC 或 SPI）                        |
| 20~21    | mpu6050         | 六轴姿态传感器 MPU6050 数据读取及 DMP 解算           |
| 22       | sht30           | SHT30 温湿度传感器（IIC）                            |
| 23       | eeprom          | EEPROM（如 AT24Cxx）读写操作                         |
| 24       | adr_res         | ADC 采集与电阻测量                                   |
| 25       | rtc             | 内部 RTC 实时时钟驱动                                |
| 26~27    | nrf24l01        | 2.4G 无线模块 NRF24L01 通讯实验                      |

#### 开发环境

- **IDE**：Keil MDK-ARM（uVision5）
- **调试器**：ST-Link / J-Link
- **编程语言**：C（寄存器/标准库/HAL）
- **RTOS**：RT-Thread（可选）
