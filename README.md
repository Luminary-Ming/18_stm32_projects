本项目是基于 **STM32F103RBT6** 的嵌入式外设驱动实验合集，涵盖从 GPIO、定时器、串口到 I2C/SPI 传感器、无线模块（NRF24L01）等 27 个独立工程。实验采用标准外设库、HAL 库（CubeMX）及 RT-Thread 多种开发方式，包含 DHT11/DS18B20 温湿度传感器、MPU6050 姿态解算、OLED 显示、EEPROM 读写、ADC 采集、RTC 时钟及独立/窗口看门狗等常用模块，每个工程均可独立编译运行。

代码按功能模块化组织，驱动与业务逻辑分离，并提供清晰的引脚注释与调试输出，便于快速移植和二次开发。项目适合嵌入式初学者系统学习 STM32 外设，也可作为毕业设计或竞赛的参考模板。所有工程均基于 Keil MDK 开发，配合 ST-Link 即可下载验证。
